1. mediapipe 정규화

l2 정규화와 minmaxscale이 필요하다고 생각했는데 사실 mediapipe 자체가 0~1 사이의 좌표값이고, 모델에 학습시키기 위해서는 -1~1 또는 0~1사이의 값이 나와야함. 근데 위와 같이 처리를 했을 때는 좌표값이 -3~1 사이로 나옴. 그래서 랜드마크 추출의 경우 정확하게 추출만 하는 코드로 수정

1. 슬라이딩 윈도우 기법

처음에는 영상을 30 샘플로 압축했는데 이러면 중간에 사라지는 프레임이 존재하기도 하고 연속된 동작에 대한 처리가 미흡하다고 판단함. 따라서 슬라이딩 윈도우 기법으로 30프레임을 한칸씩 이동하며 연속된 동작에 대한 샘플을 추출함
